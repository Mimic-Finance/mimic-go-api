# Mimic Finance API

### Requirements
- Docker
- Go

## How to run API
- Clone this repository
```
git clone https://github.com/Mimic-Finance/mimic-go-api.git
```

- Open `mimic-go-api` folder
```
cd mimic-go-api
```

- Run API with Docker
```
docker-compose up -d
```

- API will be running on http://localhost:3000
